# MARS

🚀 Application to get weather and pictures from mars with using NASA API

<br></br>

🐪 Technologies
 - Typescript
 - Next.JS 12
 - Tailwind.css

<br></br>

 🐰 How to install 
 ```
    $ npm install
 ```

 🐰 Run
 ```
    $ npm run start
 ```
